# Student Grade Calculator (Java)

This project is a simple Java program that calculates a student's
total marks, average percentage, and grade based on input scores.

## Features
- Calculates total and average marks
- Assigns grade automatically
- Beginner-friendly Java logic
- Console-based application

## Technologies Used
- Java
- Console application

## How It Works
The program takes marks as input, calculates the total and average,
and assigns a grade based on predefined criteria.

## Example Output
Total Marks: 450  
Average: 90%  
Grade: A
