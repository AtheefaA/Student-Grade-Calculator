# 📊 Student Grade Calculator

## 📌 Project Overview

The **Student Grade Calculator** is a Java-based application that calculates a student's grade based on the marks entered by the user. It provides an easy way to determine grades using predefined grading criteria and demonstrates fundamental Java programming concepts such as conditional statements, user input handling, and arithmetic operations.

This project is designed to help beginners understand the basics of Java programming and grade calculation logic.

---

## 🎯 Objectives

* Calculate student grades based on marks.
* Practice Java programming fundamentals.
* Implement grading logic using conditional statements.
* Improve logical thinking and problem-solving skills.

## 🚀 Getting Started

### Prerequisites

* Java JDK 8 or above

### Installation

1. Clone or download this repository.
2. Navigate to the project folder.
3. Compile the Java source file:

```bash
javac StudentGradeCalculator.java
```

4. Run the program:

```bash
java StudentGradeCalculator
```

---

## 📖 How to Use

1. Run the application.
2. Enter the student's marks.
3. The program evaluates the marks.
4. The corresponding grade is displayed.

## 📊 Sample Grading Criteria

| -------- | ----- |
